# Momentum Backend

An Express backend for the Momentum clothing app.

## Features
- Serves clothing item data through a REST API
- Handles requests from the React frontend
- Runs on port 3001
- Supports dynamic data fetching

## Tech Stack
- Node.js
- Express

## How It Works
The backend provides an endpoint that returns clothing items. The frontend fetches this data and displays it to users.
